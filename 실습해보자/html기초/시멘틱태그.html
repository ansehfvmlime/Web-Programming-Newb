Semantic HTML이란?

Semantic HTML은 태그의 이름만으로도 그 내용과 역할을 쉽게 이해할 수 있도록 만든 HTML 요소를 의미합니다. 검색 엔진과 스크린 리더가 페이지 구조를 더 쉽게 파악할 수 있습니다.

Semantic 태그의 장점

  • 구조가 명확해짐
  • 검색 엔진 최적화(SEO) 향상
  • 접근성 향상

관련 링크

  • MDN HTML 요소 문서
